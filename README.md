# Vedra - Voice Enabled Application

A React Native application with voice input capabilities.

## Project Structure

```
Vedra/
├── App.tsx
├── package.json
├── src/
│   ├── screens/
│   │   └── HomeScreen.tsx
│   ├── components/
│   │   └── MicButton.tsx
│   ├── theme/
│   │   ├── colors.ts
│   │   └── typography.ts
│   ├── hooks/
│   ├── services/
│   └── utils/
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run on iOS:
   ```bash
   npm run ios
   ```

3. Run on Android:
   ```bash
   npm run android
   ```

## Features

- Voice input via MicButton component
- Themed UI with custom colors and typography
- React Native based cross-platform app
